## Installation

first install yarn with npm

```bash
npm install --global yarn
yarn init
```
install modules

```bash
npm install
yarn add express yup config cors express mongoose pino pino-pretty dayjs bcrypt jsonwebtoken lodash nanoid
npm install ts-node
npx tsc --init
```
